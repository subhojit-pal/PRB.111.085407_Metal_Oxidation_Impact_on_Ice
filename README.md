
# Impact-of-metal-oxidation-on-ice-growth-and-melting
#This repository contains the data used for the figures within the project "Impact of Metal Oxidation on Ice Growth and Melting." The final paper was published on Febuary 7th 2025 in Physical Review B.
# Dataset: Impact-of-metal-oxidation-on-ice-growth-and-melting
---

# 1. Dataset Description

This dataset contains the numerical simulation data, figure source files, and scripts used in the study of **Impact-of-metal-oxidation-on-ice-growth-and-melting**.
